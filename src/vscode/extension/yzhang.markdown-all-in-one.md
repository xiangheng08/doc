# Markdown All in One

## 快捷键

| 快捷键                 | 命令                    |
| ---------------------- | ----------------------- |
| `Ctrl/Cmd + B`         | 切换粗体                |
| `Ctrl/Cmd + I`         | 切换斜体                |
| `Alt+S (仅Windows)`    | 切换删除线¹             |
| `Ctrl + Shift + ]`     | 切换标题（升级）        |
| `Ctrl + Shift + [`     | 切换标题（降级）        |
| `Ctrl/Cmd + M`         | 切换数学环境            |
| `Alt + C`              | 勾选/取消勾选任务列表项 |
| `Ctrl/Cmd + Shift + V` | 切换预览                |
| `Ctrl/Cmd + K V`       | 切换侧边预览            |
