# ECharts

## 简介

ECharts（Enterprise Charts）是一个由[百度](https://home.baidu.com/)开发的开源 JavaScript 数据可视化库，后来捐赠给 [Apache 基金会](https://www.apache.org/)并孵化毕业。它基于 HTML5 Canvas，提供直观、生动、可交互、可个性化定制的数据可视化图表。

ECharts 具有以下特点：

- **丰富的图表类型**：支持折线图、柱状图、散点图、饼图、K线图、地图、雷达图、和弦图、力导向布局图等多种图表类型，满足大部分数据可视化需求。
- **高度可定制化**：通过配置项可以灵活调整图表的颜色、样式、标签等几乎所有外观元素。
- **强大的交互功能**：支持数据视图、缩放、拖拽、提示框、点击事件等丰富的交互操作，增强用户体验。
- **跨平台兼容**：能够在 PC 和移动设备上流畅运行，兼容当前绝大部分浏览器。
- **高性能渲染**：针对大数据量展示进行了优化，支持流式数据和千万级数据的前端展示。

ECharts 广泛应用于数据分析、仪表盘、信息展示、动态图表等场景，是目前最流行的数据可视化解决方案之一。
