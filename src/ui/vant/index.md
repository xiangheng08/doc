# Vant UI

Vant 是一个轻量、可定制的移动端组件库，专为 Vue.js 设计。它提供了丰富的移动端 UI 组件，包括按钮、表单、导航、布局等，帮助开发者快速构建美观的移动应用界面。

Vant 由有赞前端团队开发并维护，广泛应用于各类移动端项目中，具有良好的兼容性和稳定性。
